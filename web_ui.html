API Endpoints (for AI Agents)

GET /api/status

Get complete server state snapshot

POST /api/run_tests

Execute tests (stations, parallel, timeout, settings)

GET /api/tests/{test_id}

Get test results for specific test_id

Analyze test results with AI suggestions

POST /api/rerun

Rerun specific tests with adjusted settings

POST /api/shutdown

Graceful server shutdown

Agent SDK Example

from agent_sdk import TestServerController

# Agent starts server
with TestServerController() as server:
    # Run tests
    test_id = server.run_tests(["VR", "Flight"])

    # Wait for completion
    results = server.wait_for_completion(test_id)

    # Analyze failures
    if results['failed'] > 0:
        analysis = server.analyze_failures(test_id)

        # Apply fixes
        for fix in analysis['auto_fixable']:
            server.apply_fix(fix)
            server.rerun_test(fix['test'])

# Server automatically stops
