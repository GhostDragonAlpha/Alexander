================================================================================

Issue: "Too many overlapping movable lights"
Solution:
  - Limit to 4-5 movable lights per area
  - Set SkyLights to STATIC (they don't count!)
  - Remove excess fill lights

Issue: Black screenshots
Solution:
  - Check XR/VR mode (exit with Alt+P)
  - Trigger lighting build
  - Check viewport show flags (viewmode lit)
  - Verify materials are assigned

Issue: Swarm initialization failure
Solution:
  - Use MOVABLE lights (no Swarm needed)
  - SkyLights can be STATIC without Swarm
