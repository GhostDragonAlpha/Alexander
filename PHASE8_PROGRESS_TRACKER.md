- type: Mission category
- title: Display name (with ${variables})
- description: Mission briefing (with ${variables})
- difficulty: Easy/Medium/Hard/Very Hard/Legendary
- objectives: Array of mission goals (2-4 items)
- rewards: Credits, reputation, bonus_conditions
- requirements: Equipment and rating prerequisites
- location: Spawn type and difficulty
- time_limit: Seconds (900-3600)
- variables: Substitution arrays for replayability
- ai_config: Enemy configuration and behavior
- repeatable: Boolean flag
- faction_affected: Array of affected factions
- moral_alignment: neutral/dark tags
